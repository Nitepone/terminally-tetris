# terminally-tetris

Hackathon approved tetris cli online

## Building

Also see [docs/DEVELOP.md](docs/DEVELOP.md).

### Setup

- Make sure you have dependencies installed
  - ncurses
  - cmake

### Compiling

```bash
cd src
cmake .
make
```

## Usage

Rather than explain the usage here, just use the command help via `./server -h`
and `./client -h`.
